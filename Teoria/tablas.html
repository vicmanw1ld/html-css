Tablas en HTML

Una tabla es un conjunto de celdas donde podemos alojar distintos contenidos: imágenes,
audio, vídeo, texto, etc.
Antiguamente, era uno de los elementos fundamentales que utilizabamos para poder
maquetar nuestras webs con el contenido que queríamos tener en ellos.
Con la llegada de HTML5, se empezó a utilizar etiquetas semánticas (aside, nav,
header, etc) y contenedores divs, que nos permitía tener una mayor flexibilidad
y adaptar nuestras webs de manera mucho más dinámica y visualmente más atractivas.
Las características o estructura que forman una tabla son principalmente las siguientes:
  • table: es la que define la tabla. Tiene etiqueta de apertura y cierre.
  • tr: es la que divide las filas de nuestra tabla. Tiene etiqueta de apertura y cierra
  • td: es la que divide en celdas las finas de nuestra tabla. Tiene etiqueta de apertura y cierre

PROPIEDADES Y ETIQUETAS DE UNA TABLA

  • table: Define el inicio y el fín de una tabla
  • tr:define la fila de una tabla, apertura y cierre
  • td: define la columna o celda de una tabla, apertura y cierre
  • caption:Se usa para colocar un titulo de una tabla
  • atributo align=top/bottom podemos poner el titulo encima o debajo de la tabla
  • thead: define la cabecera de una tabla
  • tbody:Define el contenido de una tabla
  • tfoot: Define el contenido del footer
  • th:Pone en negrita como un título

COMBINACIÓN DE CELDAS EN TABLAS

En las etiquetas td y th podemos combinarlas para hacer que sea más atractiva
y para establecer una serie de características que necesitemos


Usaremos las siguientes propiedades:

  • colspan:Indica el número de filas que ocupa
  • rowspan:Indica el número de columnas que ocupa
