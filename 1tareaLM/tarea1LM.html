LENGUAJE DE MARCAS

Los lenguajes de marcado, o lenguajes de marcas, provienen o toman su nombre de la práctica tradicional de marcar los manuscritos con instrucciones de impresión en los márgenes.
Esta práctica fue dando lugar a un grupo de marcas estandarizadas, se tralada hoy día al mundo de la informática

Histograma de los lengujes de marcas

LENGUAJES DE MARCAS AÑO CARACTERÍSTICAS PRINCIPALES EVOLUCIÓN

GML

(Generalized Markup Language)		 1970 El documento recibe un formato que define qué tipo de texto es. IBM Generalized Markup Language, o lenguaje de marcas generalizado de IBM, fue desarrollado por Charles Goldfarb y Raymond Lorie

SGML

(Standard Generalized Markup Language)		 1986 Lengaje de marcado general que define un conjunto de reglas para crear documentos.

SGML

Es el precursor de todos los lenguaje de marcado modernos.
Se utiliza para crear páginas web que se pueden visulizar en un navegador web.

HTML

(HYPER TEXT MARKUP LANGUAGE)		 1991 Lenguaje de marcado para crear documentos web.
Se basa en el uso de elementos:
  • ETIQUETAS
    Corchetes angulares < >
  • ATRIBUTOS
    estructura nombre="valor"
  • CONTENIDO
    Texto propiemante dicho.

HTML

Es el más utilizado en el mundo.
Se usa para crear páginas web para verlas en un navegador, a lo largo de los años se le han ido añadiendo nuevas funcionalidades y características

XML

(Extensible Markup Language)		 1998 Lenguaje de marcado extensible que se basa en SGML.
Se usa para crear documentos estructurados.
Desarrollado y evolucionado por el W3C.
Componentes:
  • Etiquetas
  • Atributos
  • Comentarios
  • Entidades predefinidas o secciones CDATA
    (Representan caracteres propios de las marcas)

XML

La respuesta a las deficiencias surgidas en el entorno HTML fue la aparición de este lenguaje.
Se utiliza en una amplia gama de aplicaciones:
  1. Creeación de documentos
  2. Transmisión de datos
  3. Creación de aplicaciones web
