FORMULARIOS EN HTML

Los formularios en HTML son elementos que permiten a los usuarios introducir datos en un sitio web. Estos datos se pueden enviar al servidor web para su procesamiento y almacenamiento, o se pueden utilizar en el lado del cliente para provocar de alguna manera una actualización inmediata de la interfaz de usuario.


  • El elemento form define el inicio y el final del formulario.
  • El atributo action especifica la URL de la página que procesará los datos del formulario.
    En caso de de omitor la info de donde va, se enviaran los datos a la misma página actual, volviendola a cargar
  • El atributo method especifica el método HTTP que se utilizará para enviar los datos. Los métodos más comunes son post y get. get se usa cuando los datos se envíen a la propia url, post en caso contrario.
  • name Es el nombre del formulario para hacerle referencia desde javascrip o el backend
  • Los elementos de entrada (input, textarea, select, etc.) permiten a los usuarios introducir datos en el formulario.
  • Cada elemento de entrada tiene un atributo name que especifica el nombre del campo de datos.
  • El elemento submit representa el botón de envío del formulario.
  • Cuando el usuario hace clic en este botón, los datos del formulario se envían a la página especificada por el atributo action.
  • enctypeCodificación para el envío del formulario, importante para el envío de archivos
  • accept-charsetFuerza a utilizar una codificación en los parámetros de texto del formulario
  • autocomplete on/of activa o desactiva el autocompletado
  • Información de texto nombres, apellidos, direcciones... input o textarea
  • Números o cantidadesEdades, precios... input para números
  • fechas u horas fecha de nacimiento, inicios eventos... input para fechas
  • verdadero/falsosi/no, A/B, ON/OF... input type="checkbox"
  • Opción única Elegir una opción de dos o más <>input type="radio" > select
  • Varias opcionesElegir varias opciones select multiple > input type="checkbox"
